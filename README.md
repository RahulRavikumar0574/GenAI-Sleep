# GenAI Career Guidance Platform

This monorepo contains:

- `frontend`: Next.js (TypeScript) app for the user interface
- `backend`: Node.js (Express) API for authentication, quiz, and career data

## Getting Started

### Frontend

```
cd frontend
npm run dev
```

### Backend

```
cd backend
npm start
```
